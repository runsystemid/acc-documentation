## Apa itu Unit Bisnis?

Unit Bisnis adalah fitur untuk mengelola dan menambahkan bagian atau divisi perusahaan yang memiliki fungsi operasional tersendiri, termasuk pengaturan data seperti kode, nama, alamat, nomor telepon, dan penentuan apakah unit tersebut merupakan unit bisnis utama.

## Menambah Unit Bisnis Baru

1. Klik **Pengaturan** pada Profil Pengguna.
2. Pilih **Konfigurasi â†’ Unit Bisnis.**
3. Klik tombol **Tambah,** tombol ini berada di pojok kanan atas.
4. Isikan informasi utama :

| Nama Field            | Keterangan                                                                                                   |
| --------------------- | ------------------------------------------------------------------------------------------------------------ |
| **Kode**              | Untuk memasukkan kode unik yang digunakan sebagai identifikasi unit bisnis.                                  |
| **Nama**              | Untuk mengisi nama resmi unit bisnis yang akan ditambahkan.                                                  |
| **Alamat**            | Untuk memasukkan alamat lengkap lokasi unit bisnis tersebut.                                                 |
| **No. Telepon**       | Untuk mengisi nomor telepon yang dapat dihubungi dari unit bisnis.                                           |
| **Unit Bisnis Utama** | sebagai unit bisnis yang menjadi pilihan default di setiap transaksi (hanya ada bisa satu unit bisnis utama) |

5. Klik **Simpan** untuk menyimpan Anggota, **Abaikan** untuk membatalkan perubahan.

## Melihat Unit Bisnis

1. Setelah disimpan, list bagan akun akan ditampilkan sesuai gambar ini:


2. Klik titik tiga di bagian kanan informasi bagan uang. Titik tiga ini merupakan **_More Options_** yang terdiri dari pilihan **Detail** dan **Edit.**
3. Klik **Detail**, digunakan untuk melihat **Detail** informasi dari list data.

## Mengedit Unit Bisnis

1. Klik titik tiga di bagian kanan informasi. Titik tiga ini merupakan **_More Options_** yang terdiri dari pilihan Detail dan Edit.
2. Klik **Edit,** digunakan untuk merubah detail informasi dari list data.


3. Klik **Simpan** untuk menyimpan perubahan Unit Bisnis.
