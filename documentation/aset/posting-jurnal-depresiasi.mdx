## Apa itu Posting Jurnal Depresiasi?

**Posting Jurnal Depresiasi** adalah tindakan mencatat atau memposting (memasukkan) beban penyusutan untuk aset tetap ke buku besar (general ledger) pada akhir periode akuntansi. Jurnal ini mengakumulasi alokasi biaya atas pemakaian aset selama masa manfaatnya sehingga beban tersebut tercermin di laporan laba-rugi dan nilai tercatat aset di neraca berubah melalui akun _Akumulasi Penyusutan_ (contra-asset).

<Info>
  Posting jurnal depresiasi yang akan menghitung nilai depresiasi dan mengupdate nilai aset.|
</Info>

<Tip>
  - _Menu ini berfungsi untuk mengeksekusi penerbitan jurnal transaksi atas perhitungan penyusutan nilai aset berdasarkan konfigurasi yang telah disimpan pada menu depresiasi aset._
  - _Aset yang dapat dilakukan depresiasi adalah aset yang berstatus aktif dan memiliki metode depresiasi yaitu garis lurus._
  - _Untuk aset aktif yang memiliki metode depresiasi: non depresiasi, tidak ditarik pada proses posting jurnal depresiasi._
</Tip>

## _Membuat Posting Jurnal Depresiasi_

1. ****_Klik menu Aset pada navigasi menu._****
2. \_Pilih \_**_Transaksi → Posting Jurnal Depresiasi_**
3. ****_Klik tombol Tambah, tombol ini berada di pojok kanan atas._****
4. \_Isikan field pada informasi utama, \_**_Tanggal, Tahun dan Bulan._**

| _Nama Field_             | _Keterangan_                                                                                                                  |
| ------------------------ | ----------------------------------------------------------------------------------------------------------------------------- |
| _No. Dokumen_            | _Nomor unik bukti atau referensi transaksi yang menghubungkan pencatatan aset untuk pelacakan administratif._                 |
| _Tanggal_                | _Menunjukkan tanggal di mana jurnal depresiasi ini dibuat atau dicatat, secara default akan mengikuti tanggal hari ini._      |
| _Tahun_                  | _Untuk memilih tahun periode akuntansi di mana depresiasi ini dibukukan._                                                     |
| _Bulan_                  | _Untuk memilih bulan periode akuntansi di mana depresiasi ini dibukukan._                                                     |
| _Depresiasi Ke_          | _Menunjukkan urutan depresiasi yang dicatat untuk aset tersebut (misalnya, depresiasi ke-3)._                                 |
| _Nama Aset_              | _Menyatakan nama aset yang sedang didepresiasi._                                                                              |
| _Nilai Aset_             | _Menampilkan nilai perolehan awal aset atau nilai buku aset sebelum depresiasi terkini dihitung._                             |
| _Nilai Depresiasi_       | _Mengindikasikan jumlah beban depresiasi yang diakui pada periode ini._                                                       |
| _Akumulasi Depresiasi_   | _Menunjukkan total seluruh beban depresiasi yang telah diakumulasikan untuk aset ini hingga periode saat ini._                |
| _Nilai Buku Bersih Aset_ | _Merepresentasikan nilai aset setelah dikurangi akumulasi depresiasi, yaitu nilai aset yang tersisa di pembukuan perusahaan._ |

5. ****_Klik Proses, untuk memproses data._****
6. ********_Klik Simpan untuk menyimpan Posting Jurnal Depresiasi, Klik Abaikan untuk membatalkan Posting Jurnal Depresiasi._********

## Membuat Posting Jurnal Depresiasi

| Nama Aset              | Menyatakan nama aset yang sedang didepresiasi.                                                                              |
| Nilai Aset             | Menampilkan nilai perolehan awal aset atau nilai buku aset sebelum depresiasi terkini dihitung.                             |
| Nilai Depresiasi       | Mengindikasikan jumlah beban depresiasi yang diakui pada periode ini.                                                       |
| Akumulasi Depresiasi   | Menunjukkan total seluruh beban depresiasi yang telah diakumulasikan untuk aset ini hingga periode saat ini.                |
| Nilai Buku Bersih Aset | Merepresentasikan nilai aset setelah dikurangi akumulasi depresiasi, yaitu nilai aset yang tersisa di pembukuan perusahaan. |

5. Klik **Proses**, untuk memproses data.
