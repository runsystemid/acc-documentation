## Apa itu Pendaftaran Aset?

**Pendaftaran aset** adalah proses mencatat aset tetap ke dalam sistem.

<Note>
  Mengelola data Aset yang dimiliki atau dikelola oleh perusahaan yang termasuk kedalam kekayaan perusahaan.
</Note>

<Tip>
  Menu Pendaftaran Aset mencakup detail seperti nama aset, tipe, nilai perolehan, tanggal pembelian, lokasi, dan informasi penting lainnya, sebagai dasar pengelolaan dan pelacakan aset ke depannya.
</Tip>

Berikut ini adalah panduan langkah demi langkah untuk membuat, melihat dan mengedit **Pendaftaran Aset** pada aplikasi Accounting+.

## Membuat Pendaftaran Aset

1. Klik menu **Aset** pada navigasi menu.
2. Pilih **Transaksi â†’ Pendaftaran Aset**
3. Klik tombol **Tambah,** tombol ini berada di pojok kanan atas.
4. Isikan field pada informasi utama :

| Nama Field | Keterangan                                                                                        |
| ---------- | ------------------------------------------------------------------------------------------------- |
| Kode Asset | Kode unik untuk mengidentifikasi dan melacak setiap aset di sistem (wajib, tidak boleh duplikat). |
| Nama Asset | Nama deskriptif aset yang memudahkan identifikasi saat pencarian, laporan, dan verifikasi fisik.  |
| Tipe Asset | Kategori aset yang menentukan pengelompokan, akun buku besar, dan aturan penyusutan.              |
| Alamat     | Lokasi fisik atau penanggung jawab aset untuk keperluan audit, pemeliharaan, dan pengiriman.      |
| Lampiran   | File pendukung (foto, invoice, garansi) sebagai bukti kepemilikan dan riwayat perawatan           |
| Catatan    | Keterangan tambahan.                                                                              |

5. Klik **Simpan** untuk menyimpan pendaftaran aset, Klik **Abaikan** untuk membatalkan pendaftaran aset.

## Melihat Data Pendaftaran Aset

1. Setelah di **Simpan**, list pendaftaran aset akan ditampilkan sesuai gambar di bawah ini:
2. Klik titik tiga di bagian kanan informasi **Pendaftaran Aset**. Titik tiga ini merupakan **_More Options_** yang terdiri dari pilihan Detail dan Edit.
3. Klik Detail, digunakan untuk melihat detail informasi dari list data.


4. Pada tampilan detail akan terlihat: Kode Asset, Nama Asset, Tipe Asset, No. Depresiasi dan Status yang sudah di buat sebelumnya.

## Mengedit Kategori Vendor

1. Klik menu **Aset** pada navigasi menu.
2. Pilih **Transaksi â†’ Pendaftaran Aset**
3. Klik titik tiga di bagian kanan informasi pendaftaran aset. Titik tiga ini merupakan **_More Options_** yang terdiri dari pilihan Detail dan Edit.
4. Klik **Edit**, digunakan untuk mengedit atau memperbarui data yang sudah ada.
5. Masukkan Kode Asset, Nama Asset,  Tipe Asset yang baru pada field masing-masing untuk mengubah nama sebelumnya.
6. Klik **Simpan** untuk menyimpan perubahan pendaftaran aset, Klik **Abaikan** untuk membatalkan perubahan pendaftaran aset.
